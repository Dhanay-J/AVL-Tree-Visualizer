# AVL Tree Visualizer

## Introduction

AVL Tree Visualizer is a web-based application built with React and `react-d3-tree` to demonstrate AVL Tree operations such as insertion and deletion in an interactive way. It dynamically updates the tree structure and balance factors for each node.

## Features

- Insert nodes into the AVL tree.
- Delete nodes by right-clicking on them.
- Automatically balances the tree after each operation.
- Interactive visualization using `react-d3-tree`.

## Installation and Setup

To set up and run the AVL Tree Visualizer locally, follow these steps:

### **1. Clone the repository**

```sh
git clone https://github.com/Dhanay-J/AVL-Tree-Visualizer.git
```

### **2. Navigate to the project directory**

```sh
cd AVL-Tree-Visualizer
```

### **3. Install dependencies**

```sh
npm i
```

### **4. Start the development server**

```sh
npm run dev
```

## Usage

- Enter a number in the input field and click the "Insert" button to add a node.
- Double-click on a node to delete it.
- Drag and navigate the tree visualization within the window.
